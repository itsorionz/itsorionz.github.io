Chat APP - Sendbird UIKit for React and Next.js

  • Description: This Real-Time Chat Application is a high-performance communication module built using the Sendbird UIKit for React and Next.js. Designed to bridge the gap between service providers and customers, it offers a seamless, mobile-responsive chat experience that supports instant messaging, file sharing, and real-time support. By leveraging Sendbird's powerful infrastructure alongside Next.js's server-side capabilities, the app ensures secure, low-latency conversations, making it an essential tool for managing complex travel logistics and providing immediate customer assistance.
  • Github URL: Private Repository

Key Features:

  • Instant Messaging – Real-time 1-on-1 and group chat capabilities for seamless interaction between pilgrims and travel agents.
  • Rich Media & File Sharing – Send and receive documents, flight itineraries, and photos directly within the chat interface.
  • User Presence & Status – Real-time indicators showing when support staff or customers are online and active.
  • Message Moderation Tools – Built-in administrative tools to manage conversations, including blocking, muting, and message filtering.
  • Customizable UI/UX – Fully themed chat interface using Sendbird UIKit to match the branding and design language of the travel platform.
  • Unread Message Counters & Notifications – Dynamic alerts and badges to ensure users never miss an important travel update.
  • Chat History & Search – Persistent storage of all conversations, allowing users to easily retrieve past information and booking details.
  • Mobile and Web Responsive – All mobile and Web resoposive, allowing users to easily retrieve past information view

Technologies used in Project:

  • React & Next.js – Frontend framework for building a fast, scalable, and SEO-optimized user interface.
  • Sendbird SDK & UIKit – Core communication infrastructure for handling real-time chat data and UI components.
  • Node.js & WebSockets – Ensuring low-latency, persistent connections for instant message delivery.
  • Tailwind CSS
  • Sendbird API Integration
